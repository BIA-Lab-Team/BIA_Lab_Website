Open-source Tools

All our software is freely available under permissive licences. We believe in open, reproducible science.

📡
InfoSig
Information-theoretic analysis of signaling pathways

Compute mutual information, channel capacity, noise decompositions, and partial information decompositions from bulk or single-cell signaling data. Supports GPU acceleration via JAX. Integrates with AnnData/Scanpy ecosystems.

Python JAX MIT ✓ Active · v2.1
import infosig as isg
# compute mutual information from single-cell data
mi = isg.mutual_info(adata, input_key="stimulus", output_key="X_pca")
print(f"I(X;Y) = {mi:.3f} bits")
GitHub Documentation PyPI Cite Paper

SpatialFlow
Directional signaling reconstruction from spatial transcriptomics

A graph-based toolkit for inferring the direction and magnitude of cell–cell communication in spatially resolved transcriptomic datasets. Supports 10x Visium, Slide-seq, MERFISH, and Xenium inputs. Outputs ligand–receptor flow maps and tissue-level signaling graphs.

Python R interface Apache 2.0 ✓ Active · v1.3
from spatialflow import FlowGraph
fg = FlowGraph.from_visium("sample.h5ad")
fg.infer_flows(lr_database="CellChat")
fg.plot_tissue_map(min_flow=0.1)
GitHub Documentation PyPI Cite Paper

ClockSDE
Stochastic modelling of oscillatory gene expression

Simulate and fit stochastic differential equation models of coupled genetic oscillators. Includes the Kuramoto, Goodwin, and custom coupling models. Bayesian parameter inference via NUTS/HMC using NumPyro. Designed for segmentation clock, circadian, and NF-κB oscillation datasets.

Python NumPyro MIT ✓ Active · v0.9
GitHub Documentation Cite Paper

NetMotif
Network motif detection and functional classification

Fast enumeration and statistical enrichment of network motifs in gene regulatory and signaling networks. Supports directed, signed, and weighted graphs. Classifies motifs by dynamical function (pulse generator, bistable switch, oscillator) using pre-computed phase diagrams.

Python C++ core GPL-3.0 ✓ Active · v1.0
GitHub Documentation Cite Paper

GradientFit
Morphogen gradient inference from imaging data

Fit mechanistic gradient models (diffusion–degradation, planar cell polarity) to fluorescence microscopy data. Includes tools for PSF deconvolution, automated cell segmentation, and uncertainty quantification via bootstrap. Works with Fiji/ImageJ output.

Python MATLAB MIT ⚠ Maintenance
